# Deep Reinforcement Learning for Furniture Layout Simulation in Indoor Graphics Scenes

Our paper is at https://arxiv.org/pdf/2101.07462.pdf

## Installation
First install the dependencies via conda:
 * PyTorch >= 1.0.0
 * networkx
 * numpy
 * Python >= 3.6

And then run `pip install .`
